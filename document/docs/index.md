## 3DGS：快速渲染和高质量场景重建

三维高斯泼溅技术（3D Gaussian Splatting，3DGS）使用 3D Gaussian 椭球表示三维模型，每个高斯核函数都包含参数{空间位姿；协方差矩阵；颜色；透明度}，通过 2D 投影自监督优化三维高斯函数来表示场景，实现高质量场景重建和渲染，提高了训练和渲染速度。泼溅不做光线追踪，而是把椭球簇“泼溅”到二维图像平面，泼溅痕迹信息叠加后支持直接在二维坐标中读取到对应三维位置的属性，便于 2D 语义、ORB 特征等信息嵌入。高斯分布能够精确建模薄壁、杆状物等结构，避免体素方法的量化误差。该方法实现了辐射场的实时渲染，能够在较少的训练时间内实现 SOTA 级别的效果，代表了计算机图形学在处理真实场景表示和渲染技术上的彻底变革。研究结果显示，3DGS 针对多张图像或视频拍摄的场景时，允许以 1080P 分辨率进行实时（≥30FPS）的新视图渲染。显然，在经典 3D 高斯计算框架下，将基于 SfM 的离线步骤修改为通过新输入图像反投影初始化点云，并在场景点云集合上进行 3D 高斯椭球函数训练，则能大幅提升场景表征的计算效率，同步实现高速自监督学习和全场景渲染。

## 3DGS 与 SLAM 的集成

3DGS 可被集成在 SLAM 系统中一方面是因为建图流程的一致性，即视觉 SLAM 系统通常包含从二维到三维的图像反投影和三维到二维重投影优化相机位姿的环节，这与 3DGS 中基于多视图的 SfM 三维重建和三维到二维的沿射线的可微分体渲染相契合；另一方面则是因为 3DGS 中的高斯椭球可视为带有颜色、深度和不透明度等属性的广义特征点，可为相机位姿估计提供约束更强的密集型梯度优化，从而提高 SLAM 跟踪线程的鲁棒性和精度，同时高斯椭球的自适应密度控制，即对于欠重建区域克隆高斯椭球并沿着梯度方向移动以覆盖几何体，对于过重建区域拆分高斯椭球以更好地拟合细粒度细节，支持 SLAM 建图线程的地图稠密化。 -->
